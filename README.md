# Computer Vision  with OpenCV

## Overview

This repository contains a collection of **Computer Vision Tutorial** implemented using **OpenCV** and **NumPy** in Python. The projects cover fundamental image processing, feature detection, object tracking, machine learning, and frequency-domain analysis.



---

## Topics Covered

- Image Reading and Writing
- Video Processing
- Drawing Shapes and Text
- Mouse Events and Trackbars
- Image Arithmetic and Bitwise Operations
- Image Blending
- Thresholding and Adaptive Thresholding
- Histograms and Histogram Back Projection
- Geometric, Affine, and Perspective Transformations
- Image Smoothing and Filtering
- Morphological Transformations
- Edge Detection
- Contour Detection
- Template Matching
- Hough Line Detection
- Corner Detection
- Image Pyramids and Pyramid Blending
- Feature Detection and Matching (SIFT + FLANN)
- Object Tracking (MeanShift, CamShift, Homography)
- Optical Flow (Lucas-Kanade)
- Background Subtraction
- Fourier Transform
- Face Detection using Haar Cascades
- K-Nearest Neighbors (KNN)
- Handwritten Digit Recognition

## Installation

### Clone the Repository

```bash
git clone https://github.com/2RAJARYAN/opencv.git
```

### Install Dependencies

```bash
pip install opencv-python opencv-contrib-python numpy matplotlib
```

---

## Dependencies

- Python 3.x
- OpenCV (`opencv-python`)
- OpenCV Contrib (`opencv-contrib-python`)
- NumPy
- Matplotlib
